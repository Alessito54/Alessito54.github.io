# Alessito54 - CV Portfolio

A personal CV/portfolio website built with **Angular 21**.

## Features

- 📋 **About Me** – Short personal summary
- 💼 **Work Experience** – Timeline of professional experience
- 🎓 **Education** – Academic and certification history
- 🛠️ **Skills** – Visual skill bars by category
- 📬 **Contact** – Contact information cards

## Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
ng serve
```

Navigate to `http://localhost:4200/` to view the app.

## Build

```bash
ng build
```

Artifacts are output to the `dist/cv-portfolio/` directory.

## Tests

```bash
ng test
```
